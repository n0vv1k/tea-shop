# 🍵 TEA — Premium Digital Tea Shop

Современный многостраничный веб-сайт для магазина элитного чая. Проект сочетает в себе эстетику футуристического минимализма и передовые технологии фронтенд-разработки.

![Preview](public/images/bg.jpg)

## 🏗 О проекте

**TEA** — это полноценное веб-приложение (E-commerce Web App), разработанное с упором на визуальный опыт и производительность. Сайт предлагает пользователю не просто выбор товара, а погружение в атмосферу чайной культуры через плавные анимации и продуманный UI.

**Ключевой функционал:**

- **Каталог**: Динамическая фильтрация по сортам чая и мгновенный поиск.
- **Product Page**: Индивидуальные страницы товаров с детальным описанием и макро-фотографиями.
- **Корзина**: Интуитивно понятный интерфейс управления покупками.
- **Анимации**: Использование сложных переходов между страницами для создания бесшовного опыта.

## 🤖 Контент и Дизайн (AI Collaboration)

Визуальная составляющая проекта создана в тандеме с искусственным интеллектом:

- **Visual Assets**: Все изображения товаров, фоны и элементы интерфейса сгенерированы нейросетями специально для этого проекта.
- **Design Concept**: Использование стиля "Liquid Glass" (стеклянные панели) и темной темы с неоновыми акцентами.

## 🛠 Технологический стек

- **Core:** [React 19](https://react.dev/)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://greensock.com/gsap/)
- **Routing:** [React Router 7](https://reactrouter.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Styling:** Modern CSS (Fluid Typography, CSS Clamp, CSS Variables)

## 🚀 Как запустить

1.  **Клонирование:**
    ```bash
    git clone [https://n0vv1k.github.io/tea-shop/](https://n0vv1k.github.io/tea-shop/)
    cd tea-project
    ```
2.  **Установка зависимостей:**
    ```bash
    npm install
    ```
3.  **Запуск в режиме разработки:**
    ```bash
    npm run dev
    ```
